React中的生命周期钩子函数

发表于 2020-11-20

生命周期

生命周期是组件从实例化到渲染到最终被销毁的整个过程，在此周期中会有很多可以调用的事件，也称钩子函数。

生命周期的3个状态

Mounting:将组件插入到DOM中。

Updating：将组件更新到DOM中。

Unmointing：将组件移除DOM中。

生命周期中的钩子函数(方法，事件)

compontWillMount：组件即将要渲染

compontDidMount：组件渲染完毕

compontWillReceiveProps：组件将要受props数据

shouldComponentUpdate：组件接收到新的state或者props。返回布尔值

compontWillUpdate：组件即将更新

compontDidUpdate：组件已经更新

componeWillUnmount：组件将要被卸载
